# Aktuális állapot

A live GitHub Pages dashboard már elérhető itt:
https://alexanderjudin-cpu.github.io/feladatkozpont-user-dashboard-static/

Jelenleg különálló frontend. Nem része a fő Retool appnak.

A jelenlegi GitHub index.html tartalmazza:

- kattintható top tabok
- board/list/calendar/automation/settings nézetek
- kártya drag & drop
- detail panel
- user monogram stack
- board members drawer
- user szűrés
- task notification placeholder
- AI gyorsnézet / AI agent placeholder
- postMessage bridge

Megjegyzés: a `frontend/index.local_snapshot.html` egy lokális snapshot. A legfrissebb éles fájl a GitHub repo `main/index.html` fájlja.
